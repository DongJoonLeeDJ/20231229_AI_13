1. 원의 반지름 r 값을 입력받고, "원의 넓이와 원의 둘레값" 출력하기
단 원주율은 PI=3.14로 할 것
원의 넓이 : r * r * 3.14, 원의 둘레 : 2 * r * 3.14

2. byte를 입력받고 그에 해당하는 bit값 출력하기
이 것도 상수를 이용할 것(BYTE = 8)
입력 : 1 -> 출력 : 8
입력 : 5 -> 출력 : 40

3. 정수를 입력받고 그에 대한 분과 초를 출력(단 3600미만까지 입력받는다고 가정함)
MIN = 60 이라는 상수를 활용할 것
입력 : 123 -> 출력 : 2분 3초

4. 추가 문제
4-1)현재 연도와 태어난 연도를 입력하여 나이를 구한다.
4-2)MONTH=12라는 상수를 활용하여 내가 생후 몇 개월인지 출력한다.
